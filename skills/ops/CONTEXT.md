# ops — Catalog Context

Rules, notes, and references that apply only to skills in this catalog.
(Repo-wide standards live in `.agents/knowledge/skill-quality.md`.)

## Requirements

- Skills here are **not invoked directly by users**. Their descriptions
  should reflect being triggered from within other workflows or by agent
  decision, not by user trigger phrases.
- Each skill must state its preconditions (required tools, environment
  state) explicitly, since it may run in the middle of another workflow
  without a user available to fill gaps.
