# Chinese academic register

Load this when the manuscript is in Chinese.

## Colloquial → objective

- “我们发现X” → “实验结果表明X” / “数据显示X”；保留“我们”仅用于交代作者的
  选择（“我们采用”“我们将范围限定为”）。
- “很多/特别/非常好” → 具体数量、幅度或对比（“提升 23%”）。
- “弄清楚” → “厘清/明确”；“用了” → “采用”；“做实验” → “开展实验”。
- 语气词与口语连接（“其实”“说白了”“当然啦”）全部删除。
- 设问句和面向读者的称呼（“读者可能会问”）改为陈述句。

## 欧化长句修复

- 多重“的”定语堆叠拆为短句：“基于深度学习的、面向边缘设备的、低功耗的
  推理框架” → “该推理框架基于深度学习，面向边缘设备，功耗较低”。
- “进行/开展” + 动名词还原为动词：“对数据进行分析” → “分析数据”（保留
  “开展实验”这类固定搭配）。
- 少用“被”字句；中文学术惯用主动语序或受事主语句（“该问题已得到解决”）。
- “作为...，...” 开头的英译句式多数可以直接删去“作为”短语。

## 术语与表达纪律

- 术语一次定名，全程一致；不为避免重复而换用近义词。首次出现给出英文原词
  （“注意力机制（attention mechanism）”），此后只用中文定名。
- “之一”慎用：“最重要的方法之一”多数场合直接改为明确判断或列举。
- 数字、单位使用半角；正文标点使用全角；变量与代码用半角并保持字体约定。
- 空洞评价词（“具有重要意义”“效果显著”“影响深远”）替换为可核查的具体
  陈述；程度副词（“非常”“极大地”）用数据替代。
