# Japanese academic register

Load this when the manuscript is in Japanese.

## Style base

- Body text uses だ・である体 throughout; です・ます appearing anywhere in
  the body is a register break (acknowledgements may differ by venue).
- Prefer 漢語 verbs in formal positions (行った、示した、得られた) but do
  not stack ornamental 漢語 where a plain verb reads precisely.
- Omit recoverable subjects; 「本研究では」「本稿では」 anchor the agent
  once per section, not per sentence.

## Colloquial → objective

- 「〜と分かった」「〜が見つかった」 → 「実験結果から〜が示された」
  「〜が明らかになった」.
- 「すごく/かなり/とても」 → 具体的な数値・幅（「23%向上した」）.
- 「〜と思う」「〜と感じる」 → 「〜と考えられる」「〜と推察される」 —
  and use each hedge once; 「〜と考えられるかもしれない」 is a double
  hedge.
- 「〜することができる」 → 「〜できる」; 「〜という」 padding deleted
  where the referent is already definite.
- Reader address and rhetorical questions are out of register; state
  conclusions declaratively.

## Terminology and expression discipline

- One term, one concept: once 「遅延」 is chosen, do not rotate with
  「レイテンシ」 for variety. On first use give the English original in
  parentheses — 「注意機構（attention mechanism）」 — then use the fixed
  Japanese term.
- Katakana loanwords only where the field's established term is katakana;
  otherwise prefer the standard 漢語 term.
- Particles: check は/が (topic vs subject), に/で (location of action),
  より/から (comparison vs origin) — the most common non-native and
  generated-text slips.
- Full-width 、。 in body text; half-width only inside formulas, code,
  numbers, and Latin names. 「――」 and 「…」 do not belong in academic
  prose.
- Empty evaluations (「重要な意義を持つ」「顕著な効果を示す」 with no
  numbers) are replaced by checkable statements.
